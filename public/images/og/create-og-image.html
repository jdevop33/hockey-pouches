Create Your OpenGraph Image

This page displays how your OpenGraph image will look when shared on social media platforms like Facebook, Twitter, and LinkedIn.

To create the final image:

  1. Take a screenshot of the image above (use your browser's screenshot tool or an OS screenshot utility)
  2. Make sure to capture exactly 1200×630 pixels
  3. Save the screenshot as "hockey-pouches-og.png" in the /public/images/og/ directory
  4. Update your layout.tsx file to use this new image for OpenGraph and Twitter cards

Note: For the best results on social media platforms, use a PNG image rather than SVG.
